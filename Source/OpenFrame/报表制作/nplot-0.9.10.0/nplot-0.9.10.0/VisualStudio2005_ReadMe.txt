The NPlot project is currently targeting the .Net Framework version 1.1.

In order to open this project in Visual Studio 2005 one must first install MSBee.

MSBee is a small tool from Mircosoft that allows Visual Studio 2005 projects to target the v1.1 framework.

To download MSBee go to:
http://msdn.microsoft.com/vstudio/downloads/tools/msbee/default.aspx
